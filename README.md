# FHCookieGuard

## Usage

```js
import { CookiePopup } from '@freshheads/cookie-guard'

const App = () => {
    return (
        <div>
            <CookiePopup />
        </div>
    );
};
```

We provide some basic styles but you can also use your own.
Import them as global css.
```css
@import '@freshheads/cookie-guard/dist/style.css';
```
